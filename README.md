# Online College Search Platform

## 📌 Project Overview
The Online College Search Platform is a web application that helps students easily search and discover colleges in India. The platform provides information about top colleges offering B.Tech, M.Tech, and MBA programs along with basic details.

## 🎯 Objective
The main objective of this project is to provide a simple and user-friendly platform where students can explore different colleges and courses in one place.

## ⚙️ Technologies Used
Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Node.js
- Express.js

Database:
- JSON Dataset (College Information)

## 🏗 System Architecture
User → Web Interface → Node.js Server → College Dataset (JSON) → Display Results

## ✨ Features
- Search colleges based on courses
- View list of colleges
- Display college details
- Simple and user-friendly interface
